# 🌟 Zenith - Stress Management App

**AI-powered wellness tool for developers & students**

Built by **Rishabh** — First Year BCA Student

---

### ✨ Features

- Live Stress Orb with real-time feedback
- Animated ECG Waveform (Canvas)
- Guided Breathing Exercises (4-7-8, Box, Coherent, etc.)
- Weekly Trends & Charts (Chart.js)
- Mood Tracker + Smart Journal
- Fully responsive beautiful dark UI

### 🚀 How to Use

1. Open `index.html` in any browser
2. No installation required

### 🛠 Tech Stack

- HTML5 + CSS3 + Vanilla JavaScript
- Canvas API
- Chart.js
- Single file architecture

### 🤖 Development Story

As a first-year BCA student with limited programming background, I built this entire app with the help of Claude AI. This project helped me learn modern frontend development, animations, state management, and how to ship polished products quickly.
